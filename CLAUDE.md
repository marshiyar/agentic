## Core Insight

Unix tools work because the philosophy is embodied, not documented.

## The Disintermediation Principle

Keep frontier models in the critical path. Build infrastructure that amplifies model capabilities, not replaces them.

When new models drop, apps built this way get better automatically. Apps with logic baked into code just get nicer explanations of the same outputs.

**Do:** MCP tools that give models capabilities (query APIs, invoke functions, discover resources)

**Don't:** Consensus algorithms, prompt management systems, hardcoded reasoning flows
